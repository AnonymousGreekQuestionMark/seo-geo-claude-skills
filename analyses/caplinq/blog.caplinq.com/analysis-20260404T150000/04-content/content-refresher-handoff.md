### Refresh Protocol for CAPLINQ Blog

**Quarterly refresh triggers:**
- ECHA SVHC list updates (quarterly) → update any REACH-related posts
- New product introductions from supplier partners → update product comparison posts
- Regulatory changes (PFAS, RoHS, REACH) → update compliance posts

**Annual refresh triggers:**
- Thermal conductivity benchmarks → update TIM posts
- Market size statistics → update any industry market posts
- Customer case studies → add new application examples

**Post Update Checklist:**
1. Update all statistics with current year data
2. Check product availability — remove discontinued products
3. Add any new product options since original publication
4. Add "Last Updated: [Month Year]" to post header
5. Re-test schema markup in Google Rich Results Test
6. Add or update FAQPage schema if Q&A content present
7. Add "Key Takeaways" box if not present
8. Verify all internal links still go to correct URLs
