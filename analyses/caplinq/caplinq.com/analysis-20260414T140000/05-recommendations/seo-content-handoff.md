## Content Templates

### Hero Product Page Template
```
1. H1: [Primary Keyword]
2. Hero section: Definition + value prop (100 words)
3. What is [keyword]? (200 words, target featured snippet)
4. Types/Categories (300 words + comparison table)
5. Applications by Industry (400 words + visuals)
6. Selection Guide (300 words + decision tree)
7. Product Listings (from database)
8. FAQ Section (5-10 questions, with FAQ schema)
9. Related Products/Resources
10. Contact CTA
```

### Blog Article Template (AI-Optimized)
```
1. H1: [Long-tail keyword as question]
2. TL;DR / Key Takeaways (3-5 bullets)
3. Definition paragraph (target AI extraction)
4. Detailed explanation with H2s
5. Comparison table (if applicable)
6. Expert insights (with author credentials)
7. External citations (2-3 authoritative sources)
8. FAQ section (3-5 questions)
9. Related reading
```

## Content SEO Specifications

| Element | Specification |
|---------|---------------|
| Title tag | Primary keyword + brand | 50-60 chars |
| Meta description | Action-oriented + keyword | 150-160 chars |
| H1 | Exact or close keyword match | 1 per page |
| H2s | Related keywords/topics | 3-7 per page |
| Images | Descriptive alt text | All images |
| Schema | Minimum: BreadcrumbList + 1 content type | All pages |
| Internal links | 5-10 contextual links | Per page |
| External links | 1-3 authoritative sources | Per page |
